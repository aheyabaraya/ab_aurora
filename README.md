# AB_Aurora

AB_Aurora is a Brand Persona Director Agent.  
This repository is bootstrapped for v0 using Next.js + TypeScript + Tailwind.

## Quick start
```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

Canonical source root is this repository root (`ab_aurora`).

## Quality gates
```bash
pnpm lint
pnpm typecheck
pnpm test:agent
```
